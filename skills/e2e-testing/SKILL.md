# End-to-End Testing

Run the smallest meaningful user journey in the project's supported environment. Avoid inventing infrastructure. State prerequisites, commands, outcome, and any blocked scenario; do not treat unavailable E2E coverage as passing evidence. Give quality verified start, URL or port, and cleanup details for `.agents/artifacts/local-test.md`, or state what prevents them.
